# 🎮 Faiq's Gaming Hub

## 📌 Overview
Faiq's Gaming Hub is a C++ based multi-game project developed using the SFML (Simple and Fast Multimedia Library). It combines four classic games into a single interactive gaming hub with smooth UI, animations, sound effects, and scoring systems.

This project was built to strengthen Object-Oriented Programming (OOP) concepts, game development logic, and SFML graphics handling.

---

## 🕹️ Games Included
This Gaming Hub contains the following four fully functional games:

- ✖️ Tic Tac Toe  
- 🐍 Snake and Ladder  
- 🏓 Ping Pong  
- 🚀 Space Shooter  

---

## ✨ Features

- 🎮 Four games in one unified gaming hub  
- 🔊 Sound effects and background music  
- 🏆 Individual leaderboards for each game  
- 🌍 Overall global leaderboard system  
- 🎨 Smooth animations using SFML  
- ⚡ Responsive and interactive controls  
- 🧠 Object-Oriented Programming (OOP) based structure  
- 💾 Score tracking and competitive gameplay  

---

## 🛠️ Technologies Used

- C++  
- SFML (Simple and Fast Multimedia Library)  
- Object-Oriented Programming (OOP)
